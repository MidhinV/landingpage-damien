# Agent Images

Place the agent images from the PDF slides in this directory with the following naming convention:

- `data-reactivation.jpg` (or .png) - Data Reactivation AI Agent
- `receptionist.jpg` (or .png) - AI Receptionist Agent
- `sales.jpg` (or .png) - AI Sales Agent
- `admin.jpg` (or .png) - Admin AI Agent
- `customer-support.jpg` (or .png) - Customer Support AI Agent
- `client-intake.jpg` (or .png) - Client Intake AI Agent
- `invoice-tax.jpg` (or .png) - Invoice, GST & Tax AI Agent
- `onboarding.jpg` (or .png) - OnboardAI

## Image Requirements

- Recommended size: 1200x800px or similar aspect ratio (3:2)
- Format: JPG or PNG
- The images will be displayed in the agent detail modals

## How to Extract Images from PDF

1. Open the PDF in a PDF viewer or image editor
2. Export each agent slide as an image
3. Save with the corresponding filename listed above
4. Place all images in this `/public/agents/` directory

The images will automatically appear in the agent modals once added.
