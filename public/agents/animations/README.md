# Animated Agent Illustrations

Place animated illustrations for each agent in this directory. These should be friendly, professional illustrations similar to the handshake example.

## Supported Formats

- **GIF** (recommended) - Simple animated GIFs work best
- **Lottie JSON** - For more complex animations (requires Lottie player library)
- **Animated SVG** - For vector-based animations

## File Naming

Name your animated illustrations to match the agent IDs:

- `data-reactivation.gif` - Data Reactivation AI Agent
- `receptionist.gif` - AI Receptionist Agent  
- `sales.gif` - AI Sales Agent
- `admin.gif` - Admin AI Agent
- `customer-support.gif` - Customer Support AI Agent
- `client-intake.gif` - Client Intake AI Agent (handshake illustration example)
- `invoice-tax.gif` - Invoice, GST & Tax AI Agent
- `onboarding.gif` - OnboardAI

## Illustration Style Guidelines

Based on the handshake example, illustrations should:
- Be friendly and professional
- Use flat design style
- Include people/characters when relevant
- Use a light, modern color palette
- Show the agent's purpose visually (e.g., handshake for client intake)
- Be approximately 192px Ã— 128px or similar aspect ratio

## Where to Get/Create Animations

1. **Fiverr/Upwork** - Search for "business illustration animation" or "animated GIF illustration"
2. **AI Image Generators** - DALL-E, Midjourney, Stable Diffusion (then animate with tools)
3. **Animation Tools** - After Effects, Figma (export as GIF), Lottie
4. **Illustration Libraries** - unDraw, Storyset (some have animations)
5. **Freelance Illustrators** - Hire someone to create custom animations

## Example Concepts

- **Client Intake**: Handshake between two people (like your example)
- **Receptionist**: Person answering phone with friendly expression
- **Sales**: Person presenting or closing a deal
- **Admin**: Person organizing documents or scheduling
- **Customer Support**: Person helping another person
- **Data Reactivation**: Person reconnecting with old contacts
- **Invoice/Tax**: Person with calculator or financial documents
- **Onboarding**: Person being welcomed or trained
